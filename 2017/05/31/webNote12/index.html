ul篇

整个五子棋的格局由canvas写成，之前对canvas只是浅浅的了解了一下
那么今天就来学习一下canvas！

canvas画布

首先在html里加一行代码

这就是一张200x200的画布了
之后就是js的事情了

Canvas - 线条

我们来画条对角线

stroke这个方法就是用于线条。

现在我们画一个3x3的棋局

通过一个for循环完成对线条的绘制，那么15x15的五子棋棋盘就是通过这个方法实现的。
通过strokeStyle还可以对线条颜色进行更改

Canvas - 文字
当然 canvas也可以让我们写入文字进去

这里实现的是一个水平方向和垂直方向上为棋盘加辅助数字功能
其中i就是数字0-14
font - 定义字体
fillText(text,x,y) - 在 canvas 上绘制实心的文本
strokeText(text,x,y) - 在 canvas 上绘制空心的文本

Canvas - 图片

把一幅图像放置到画布上, 使用以下方法:
drawImage(image,x,y)
其中第四第五参数可选，为图片的宽高

Canvas - 渐变

为了让棋子美观，用了圆渐变

  • createRadialGradient(x,y,r,x1,y1,r1) - 创建一个径向/圆渐变
  • createLinearGradient(x,y,x1,y1) - 创建线条渐变

当我们使用渐变对象，必须使用两种或两种以上的停止颜色。
addColorStop()方法指定颜色停止，参数使用坐标来描述，可以是0至1.
使用渐变，设置fillStyle或strokeStyle的值为 渐变，然后绘制形状，如矩形，文本，或一条线。

HTML5 canvas 元素用于图形的绘制，通过脚本 (通常是JavaScript)来完成.
canvas标签只是图形容器，你必须使用脚本来绘制图形。
你可以通过多种方法使用Canva绘制路径,盒、圆、字符以及添加图像。
